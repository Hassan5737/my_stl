# my_stl

A modern C++ project focused on reimplementing common utilities and building new ones from scratch in order to deeply understand algorithms, data manipulation, and library design.


this is a learning-focused implementation that explores how such utilities can be built manually with clean and maintainable code.

## Philosophy

Instead of relying directly on existing library utilities, this project focuses on:

- Reimplementing commonly used functions from scratch
- Improving algorithmic thinking and low-level logic
- Practicing clean library design in C++
- Continuously optimizing and improving implementations over time

Each function in this library may evolve through multiple versions as performance and design improve.

## Goals of the Project

- Strengthen problem-solving and algorithmic thinking
- Understand how libraries are designed internally
- Practice writing reusable and maintainable C++ code
- Experiment with creating new utilities and algorithms

Some functions may replicate common utilities, while others may be completely new ideas built from scratch.

## Continuous Improvement

This library is intentionally iterative.

Functions may be:

- Refactored
- Optimized
- Redesigned
- Replaced with better implementations

as understanding and experience grow.

## Example Direction

Possible categories of utilities that may appear in this library:

- String utilities
- Numeric utilities
- Custom algorithms
- Data manipulation helpers
- Experimental functions
